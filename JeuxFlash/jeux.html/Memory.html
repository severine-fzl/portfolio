Jeu de mémoire

Description

Exercez votre mémoire en retrouvant les couples d'images identiques. Pour cela, vous devez cliquer sur deux cartes succéssivement.

Si vous reformez correctement les couples, les images restent apparentes. Si vous vous trompez, les cartes redeviennent faces cachées.

Le jeu se termine lorsque toutes les cartes sont retournées.
